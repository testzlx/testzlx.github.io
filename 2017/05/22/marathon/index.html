marathon部署系统

简介

  • 使用组件: docker ，mesos，marathon,服务发现，分布式日志收集，docker registry，gitlab，jenkins

优势

  • 简单易用，真正做到一键部署，提升服务部署效率
  • 灵活扩展，服务实现自动扩缩容，故障自动拉起，保证服务稳定运行
  • 对应用业务侵入几乎为0，用户只需提供日志目录，开放端口，所需cpu，mem，对外使用域名即可
  • 安全可靠，资源高度隔离，用户独享计算资源，容器运行负载实时监控、快速迁移
  • 高效，镜像极速上传下载海量容器秒级启动,使用容器服务进行构建、打包集成
  • 集群管理：对用户透明，不影响线上服务。简单快速，集群规模大小控制随心所欲
