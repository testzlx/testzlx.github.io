##常见配置

  • git别名配置
    对于status、add、commit以及checkout这种常见命令，给它们一个缩写非常有用。
    将下述内容加入您$HOME文件夹下的.gitconfig文件
    文件名: .gitconfig

一旦在.gitconfig文件中定义了别名，您就可以在需要使用”git checkout”时通过键入”git co”命令代替。 同样可以使用”git st”代替”git status”，以及”git ci”代替”git commit”。
最后，”git hist”允许您避免输入长长的log命令。

  • shell别名配置

用go缩写来替换git checkout非常有意思。它允许我通过键入”go”来检查一个特定的分支。
